# CrawlMind 🤖

AI-powered Chrome 插件 - 智能网页数据提取工具

## 简介

CrawlMind 是一款基于 DeepSeek LLM 的 Chrome 扩展程序，能够智能分析网页结构并提取用户需要的数据。只需描述你想要的数据类型，AI 会自动分析页面层级并提取。

## 功能特性

- 🧠 **智能层级分析** - 自动分析 HTML 结构，找出包含目标数据的层级路径
- 🔄 **分层提取策略** - 先分析结构，再提取数据，循环往复直到找到所需内容
- 📊 **自动格式化输出** - 根据数据类型自动格式化输出（商品名称、评论、价格等）
- 💭 **思考过程可视化** - 实时显示 AI 的分析过程和决策步骤
- 🌐 **全网页支持** - 支持任意网页的数据提取

## 安装

1. 克隆仓库到本地
2. 打开 Chrome 扩展管理页面 (`chrome://extensions/`)
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择项目文件夹

## 使用方法

1. 点击浏览器工具栏的 🤖 图标打开浮窗
2. 在输入框中描述你想提取的数据，例如：
   - "提取页面的商品名称"
   - "提取所有评论内容"
   - "提取商品价格"
3. 点击发送，AI 会自动分析并提取数据

## 工作原理

```
1. 获取页面结构（框架结构 + 完整内容）
      ↓
2. LLM 分析层级结构，判断哪个层级可能包含目标数据
      ↓
3. 如果找到目标层级 → 根据该层级提取数据
      ↓
4. 如果没找到 → 循环往下分析，每 20000 字符深挖一次
      ↓
5. 找到数据后 → 格式化输出（商品1、评论1 等）
```

## 技术栈

- Chrome Extension (Manifest V3)
- DeepSeek API (LLM)
- JavaScript

## 配置

首次使用需要配置 DeepSeek API Key：

1. 获取 DeepSeek API Key: https://platform.deepseek.com/
2. 在插件浮窗中输入 API Key 并保存

## 文件结构

```
CrawlMind/
├── manifest.json          # 插件配置文件
├── background/
│   └── background.js      # 后台脚本（LLM调用、提取逻辑）
├── content/
│   └── analyzer.js       # 内容脚本（网页结构提取）
├── popup/
│   ├── popup.html        # 弹出窗口
│   └── popup.js         # 弹出窗口逻辑
├── icons/                # 插件图标
└── lib/                 # 工具库
```

## 注意事项

- 需要有效的 DeepSeek API Key
- 部分动态加载的页面可能需要等待加载完成
- 请遵守网站的 robots.txt 和使用条款
