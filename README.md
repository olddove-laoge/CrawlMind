# CrawlMind 🤖

AI-powered Chrome/Edge 插件 - 智能网页数据提取工具

## 简介

CrawlMind 是一款基于 DeepSeek LLM 的 Chrome/Edge 扩展程序，能够智能分析网页结构并提取用户需要的数据。只需描述你想要的数据类型，AI 会自动分析页面层级并提取。

## 功能特性

- 🧠 **智能层级分析** - 自动分析 HTML 结构，找出包含目标数据的层级路径
- 🔄 **分层提取策略** - 先分析结构，再提取数据，循环往复直到找到所需内容
- 📊 **自动格式化输出** - 根据数据类型自动格式化输出（商品名称、评论、价格等）
- 💭 **思考过程可视化** - 实时显示 AI 的分析过程和决策步骤
- 🌐 **全网页支持** - 支持任意网页的数据提取
- 🔍 **Shadow DOM 支持** - 支持遍历 Shadow DOM 结构（如 B 站评论）
- ✅ **用户确认机制** - 提取数据后展示给用户确认，确保准确性
- 🔄 **智能反思机制** - 连续失败时自动触发反思，重新分析页面结构
- 💾 **选择器持久化** - 保存已验证的选择器，下次同类型页面可直接使用
- 📥 **一键导出** - 支持导出 CSV/Excel 格式
- 🎯 **精确选择器生成** - 基于 DOM 定位生成更精准的 CSS 选择器

## 安装

1. 克隆仓库到本地
2. 打开 Chrome/Edge 扩展管理页面 (`chrome://extensions/` / `edge://extensions/`)
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择项目文件夹

## 使用方法

1. 点击浏览器工具栏的 🤖 图标打开浮窗
2. 在输入框中描述你想提取的数据，例如：
   - "提取页面的商品名称"
   - "提取所有评论内容"
   - "提取商品图片链接"
3. 点击发送，AI 会自动分析并提取数据
4. 提取完成后可导出为 CSV/Excel

## 工作原理

```
1. 获取页面结构（框架结构 + 完整内容，包含 Shadow DOM）
      ↓
2. 查找已保存的选择器（域名 + 需求匹配）
      ↓
3. 验证选择器有效性 → 直接使用 或 回退到正常分析
      ↓
4. LLM 分析层级结构，判断哪个层级可能包含目标数据
      ↓
5. 如果找到目标层级 → 根据该层级提取数据
      ↓
6. LLM 验证数据是否真实存在（部分存在即通过）
      ↓
7. 展示给用户确认
      ↓
8. 用户确认 → 继续获取更多数据 → 保存选择器（可选）
   用户拒绝 → 分析拒绝原因 → 反思机制 → 重新寻找
      ↓
9. 提取完成 → 导出数据
```

## 选择器持久化

- 爬取完成后可选择保存当前选择器
- 保存时自动生成标题（如"淘宝商品名称选择器"）
- 下次爬取相同类型页面时，自动查找并验证已保存的选择器
- 多个选择器时由 LLM 自动选择最合适的

## 反思机制

当提取失败时，系统会自动触发反思：

- **连续 5 次失败** 或 **累计 10 次失败** 时触发
- LLM 重新分析页面结构
- 提供新的层级路径和建议的偏移量
- 跳过无效区域，直接跳转到可能包含数据的位置

## 技术栈

- Chrome Extension (Manifest V3)
- DeepSeek API (LLM)
- JavaScript

## 配置

首次使用需要配置 DeepSeek API Key：

1. 获取 DeepSeek API Key: https://platform.deepseek.com/
2. 在插件浮窗中输入 API Key 并保存

## 文件结构

```
CrawlMind/
├── manifest.json          # 插件配置文件
├── background/
│   └── background.js      # 后台脚本（LLM调用、提取逻辑）
├── content/
│   └── analyzer.js        # 内容脚本（网页结构提取、Shadow DOM遍历）
├── popup/
│   ├── popup.html         # 弹出窗口
│   └── popup.js           # 弹出窗口逻辑
├── data/
│   └── selectors.json     # 选择器持久化存储
├── icons/                 # 插件图标
└── lib/                   # 工具库
```
